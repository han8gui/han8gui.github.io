“穷人”的思维

发表于 2017-11-03

情景：

最近在苏宁买相机，下了三笔订单
第一单 4096元 线下
第二单 3997元 线上
第三单 3999元 线上

第二单是因为第一单嫌贵，然后下的。
第三单是因为12月免息 + 单肩包

最后的结果是下了第三单后，取消第二单。第三单虽然下单，但是支付不了，最后什么都没得到。

穷人的想法是想占尽所有的好处，却不知道有些事情看似无风险，其实是高风险。

分类 以为的风险 收益 实际风险 实际收益
第一单 无 - - -
第二单 无 100元 无 100元
第三单 无 12月免息 + 单肩包 订单无法支付 无（归0）

信息不对称

以为：下单一定能够支付
实际：下单不一定能够支付成功
实际操作：第三单下单（未支付），取消第二单。
建议：同时支付两单，最后并退款。
联想：找工作的时候的骑驴找马

模型：

以为的：
本金：相机购买权
利益：12月免息 + 单肩包
风险：0%

实际的：
本金：损失
利益：0
风险：100%
额外损失：补救的解决方案的成本(时间，金钱)，最后还没解决问题。

总结

  1. 决策时往往只看到利益，而不注意风险。
  2. 信息不对称时，决策的风险是很大的。
  3. 客服的作用就是转移你的注意力，并不是帮你解决问题。（潜台词：很抱歉，但是我帮不上忙）
